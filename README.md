# Hotel Reservation System (Java)

A console-based interactive Hotel Reservation System developed using Java.

## Features
- User and Admin login
- View available rooms
- Book and cancel reservations
- Manage customer information
- Admin panel to add rooms
- Colorful console output using ANSI codes

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner
- Console-based UI

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run `Main.java`
